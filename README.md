# La Grancesela Finished Local Prototype

## What is included
- MySQL database schema and sample data
- Node.js Express backend API
- Admin frontend UI
- Public ticket form
- Date range availability feature

## Setup
1. Import `database/schema.sql` in MySQL.
2. Import `database/sample_data.sql` in MySQL.
3. Copy `backend/.env.example` to `backend/.env` and update DB_PASSWORD.
4. Run `npm.cmd install` inside `backend`.
5. Run `npm.cmd run create-admin` inside `backend`.
6. Run `npm.cmd start` inside `backend`.
7. Run `npx.cmd serve .` inside `frontend`.
8. Open the local frontend URL and go to `/live_admin.html`.
